Today, we're going to learn how to **tell time** in Arabic! This is super useful for everyday conversations, and we'll use a lot of the things you've already learned.

## Lesson

1.  **Asking "What time is it?"**
    *   To ask "What time is it?" in Arabic, we use a new question word:
        *   **كَمْ (kam)**: This means **"How many?"** or **"How much?"**
            *   *Pronunciation hint*: Short 'a' sound, like 'kam'.
    *   We combine this with a new noun:
        *   **السَّاعَةُ (As-saa'atu)**: This means **"the hour,"** **"the clock,"** or **"the watch."**
            *   *Pronunciation hint*: `السَّاعَةُ` has `الْـ` (the definite article) and the سِين (Seen) is a Sun Letter, so the 'L' is silent and the سَ gets a Shaddah. It also ends in a تَاء مَرْبُوطَة (ة), which tells us it's a **feminine** noun.
    *   So, to ask "What time is it?" we say:
        *   `كَمِ السَّاعَةُ؟` (Kam as-saa'atu?)
            *   *Notice*: The `مْ` of `كَمْ` gets a Kasra (`مِ`) for smooth pronunciation when followed by `السَّاعَةُ`. For now, just learn this as the fixed phrase.

2.  **Answering "It is [X] o'clock."**
    *   When we answer, we say "The hour" (`السَّاعَةُ`) and then use a special descriptive word (like an adjective) for the number of the hour.
    *   These descriptive words for hours are called **ordinal numbers** (like "first," "second," "third" instead of "one," "two," "three").
    *   Because `السَّاعَةُ` (the hour) is a **feminine** noun, all the ordinal numbers we use to describe it must also be **feminine** (ending with `ـَةُ`) and **definite** (starting with `الْـ`), just like we learned with adjective matching on Day 11!
    *   The structure is: `السَّاعَةُ` (The hour) + `الْـ [Feminine Ordinal Number]` (the [Ordinal Number]). The "is" is understood.

3.  **The Ordinal Hours (1 to 12):**

    Let's learn how to say each hour from one o'clock to twelve o'clock. Pay close attention to the `الْـ` and the `ـَةُ` at the end of the ordinal numbers.

    *   **One o'clock:**
        *   `السَّاعَةُ الْوَاحِدَةُ.` (As-saa'atu al-waahidatu.)
            *   (Literally: The hour, the one-feminine)
    *   **Two o'clock:**
        *   `السَّاعَةُ الثَّانِيَةُ.` (As-saa'atu ath-thāniyatu.)
            *   (Literally: The hour, the second-feminine. Note: ثَاء is a Sun Letter, so `الثَّانِيَةُ` starts with doubled ث.)
    *   **Three o'clock:**
        *   `السَّاعَةُ الثَّالِثَةُ.` (As-saa'atu ath-thālithatu.)
            *   (Literally: The hour, the third-feminine. Note: ثَاء is a Sun Letter, so `الثَّالِثَةُ` starts with doubled ث.)
    *   **Four o'clock:**
        *   `السَّاعَةُ الرَّابِعَةُ.` (As-saa'atu ar-rābi'atu.)
            *   (Literally: The hour, the fourth-feminine. Note: رَاء is a Sun Letter, so `الرَّابِعَةُ` starts with doubled ر.)
    *   **Five o'clock:**
        *   `السَّاعَةُ الْخَامِسَةُ.` (As-saa'atu al-khāmisatu.)
            *   (Literally: The hour, the fifth-feminine. Note: خَاء is a Moon Letter, so the 'L' in `الْخَامِسَةُ` is pronounced.)
    *   **Six o'clock:**
        *   `السَّاعَةُ السَّادِسَةُ.` (As-saa'atu as-sādisatu.)
            *   (Literally: The hour, the sixth-feminine. Note: سِين is a Sun Letter, so `السَّادِسَةُ` starts with doubled س.)
    *   **Seven o'clock:**
        *   `السَّاعَةُ السَّابِعَةُ.` (As-saa'atu as-sābi'atu.)
            *   (Literally: The hour, the seventh-feminine. Note: سِين is a Sun Letter, so `السَّابِعَةُ` starts with doubled س.)
    *   **Eight o'clock:**
        *   `السَّاعَةُ الثَّامِنَةُ.` (As-saa'atu ath-thāminatu.)
            *   (Literally: The hour, the eighth-feminine. Note: ثَاء is a Sun Letter, so `الثَّامِنَةُ` starts with doubled ث.)
    *   **Nine o'clock:**
        *   `السَّاعَةُ التَّاسِعَةُ.` (As-saa'atu at-tāsi'atu.)
            *   (Literally: The hour, the ninth-feminine. Note: تَاء is a Sun Letter, so `التَّاسِعَةُ` starts with doubled ت.)
    *   **Ten o'clock:**
        *   `السَّاعَةُ الْعَاشِرَةُ.` (As-saa'atu al-'āshiratu.)
            *   (Literally: The hour, the tenth-feminine. Note: عَيْن is a Moon Letter, so the 'L' in `الْعَاشِرَةُ` is pronounced.)
    *   **Eleven o'clock:**
        *   `السَّاعَةُ الْحَادِيَةَ عَشَرَةَ.` (As-saa'atu al-ḥādiyata 'asharata.)
            *   (Literally: The hour, the eleventh-feminine. This is a special fixed phrase, remember it as a whole.)
    *   **Twelve o'clock:**
        *   `السَّاعَةُ الثَّانِيَةَ عَشَرَةَ.` (As-saa'atu ath-thāniyata 'asharata.)
            *   (Literally: The hour, the twelfth-feminine. This is another special fixed phrase, remember it as a whole.)

This might look like a lot of new words, but remember, the pattern is very similar for each hour: `السَّاعَةُ` followed by its specific feminine, definite descriptive word.
